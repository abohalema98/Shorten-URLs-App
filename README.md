# Shorten-URLs-App

Link shortening application, to shorten long section links

ex: https://www.google.com.eg/ => https://cutt.ly/XBjiopZ
using: - cuttly api - node js v18.10.0 - Express - node-fetch - Angular v13 - Bootstrap.
